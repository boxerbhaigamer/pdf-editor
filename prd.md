PRD — Tournament PDF Header Automation System
1. Product Overview
Problem

During boxing tournaments, the scoring software generates hundreds of PDFs (score sheets, daily schedules, bout results, etc.).
Each PDF must have a custom tournament header, but the software cannot generate the correct header.

Currently the workflow requires:

Two laptops

Manual editing in Canva or other tools

Repeated editing for hundreds of PDFs

This process is slow, error-prone, and inconvenient.

Solution

A web-based PDF header editing system that allows users to:

Upload PDFs generated by scoring software

Apply predefined header templates

Replace headers on all pages of a PDF

Batch process hundreds of PDFs

Collaborate between two users

Download or print edited PDFs

2. Core Product Principles

Header format is fixed

Page size always A4

Header position always top of page

Header height fixed

Header exists on every page

PDF content below header must remain untouched

Manual template selection (no AI detection)

7. PDF Editing Algorithm

Library recommended: pdf-lib

Input
PDF file
Template data
Header height
Algorithm
1 Load PDF

2 Get page count

3 For each page

    a Get page width
    b Get page height

    c Draw white rectangle

        x = 0
        y = page_height - header_height
        width = page_width
        height = header_height

    d Insert template elements

        draw left logo
        draw right logo
        draw title
        draw subtitle
        draw venue
        draw dates

4 Save new PDF

5 Upload edited PDF
