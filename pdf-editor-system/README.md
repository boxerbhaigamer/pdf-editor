# Tournament PDF Header Automation System

A web-based PDF header editing system for boxing tournaments that allows users to upload PDFs generated by scoring software, apply predefined header templates, replace headers on all pages, batch process hundreds of PDFs, and collaborate between two users.

## Features

- User authentication (Admin and Editor roles)
- Tournament management
- Template management for headers
- PDF upload and management
- Header replacement on all pages
- Batch processing of multiple PDFs
- Google Drive integration for file storage

## Prerequisites

- Node.js (v14 or higher)
- PostgreSQL (v12 or higher)
- Google Drive API credentials (for production deployment)

## Setup Instructions

### Backend Setup

1. Navigate to the server directory:
   ```
   cd server
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Create a `.env` file based on `.env.example`:
   ```
   cp .env.example .env
   ```

4. Update the `.env` file with your configuration:
   - Database credentials
   - JWT secret
   - Google Drive API credentials (optional for development)

5. Set up the PostgreSQL database:
   - Create a new database
   - Run the schema.sql script to create tables:
     ```
     psql -U your_username -d your_database_name -f ../database/schema.sql
     ```

6. Start the backend server:
   ```
   npm run dev
   ```

### Frontend Setup

1. Navigate to the client directory:
   ```
   cd client
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Create a `.env.local` file with the following content:
   ```
   NEXT_PUBLIC_API_BASE_URL=http://localhost:5000/api
   ```

4. Start the frontend development server:
   ```
   npm run dev
   ```

## Google Drive API Setup

To set up Google Drive API for production:

1. Go to the [Google Cloud Console](https://console.cloud.google.com/)
2. Create a new project or select an existing one
3. Enable the Google Drive API
4. Create credentials (OAuth 2.0 Client ID)
5. Configure the OAuth consent screen
6. Add your credentials to the `.env` file:
   ```
   GOOGLE_DRIVE_CLIENT_ID=your_client_id
   GOOGLE_DRIVE_CLIENT_SECRET=your_client_secret
   GOOGLE_DRIVE_REDIRECT_URI=your_redirect_uri
   ```

## API Endpoints

### Authentication
- POST `/api/auth/register` - Register a new user
- POST `/api/auth/login` - Login user
- GET `/api/auth/me` - Get current user

### Tournaments
- POST `/api/tournaments` - Create a new tournament
- GET `/api/tournaments` - Get all tournaments
- GET `/api/tournaments/:id` - Get a tournament by ID
- PUT `/api/tournaments/:id` - Update a tournament
- DELETE `/api/tournaments/:id` - Delete a tournament

### Templates
- POST `/api/templates` - Create a new template
- GET `/api/templates` - Get all templates
- GET `/api/templates/:id` - Get a template by ID
- PUT `/api/templates/:id` - Update a template
- DELETE `/api/templates/:id` - Delete a template

### PDF Processing
- POST `/api/pdf/upload` - Upload PDF files
- POST `/api/pdf/apply-template` - Apply template to PDF
- GET `/api/pdf/download/:fileId` - Download PDF file
